# Retail Sales Analytics Dashboard

## Project Overview
The Retail Sales Analytics Dashboard is a data analytics project that analyzes retail sales data and presents insights through an interactive dashboard. The project demonstrates how raw data can be transformed into meaningful business insights using Python.

The system performs data analysis, visualization, and dashboard creation to help understand sales trends, product performance, and category-wise revenue.

---

## Objectives
- Analyze retail sales data
- Identify top-selling products
- Understand category-wise sales distribution
- Visualize sales trends
- Build an interactive analytics dashboard
